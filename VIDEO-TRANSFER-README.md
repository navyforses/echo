## ­ЪЊЂ рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ

## РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

### Node.js рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

`video-transfer.js` рЃцрЃљрЃўрЃџрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃерЃћрЃфрЃЋрЃљрЃџрЃЮрЃЌ:

```javascript
this.sourceDir = './public/videos';        // рЃгрЃДрЃљрЃарЃЮрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ
this.destinationDir = './transferred-videos'; // рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ
```

### PowerShell рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

`transfer-videos.ps1` рЃцрЃљрЃўрЃџрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃерЃћрЃфрЃЋрЃљрЃџрЃЮрЃЌ:

```powershell
param(
    [string]$SourceDir = ".\public\videos",        # рЃгрЃДрЃљрЃарЃЮрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ
    [string]$DestinationDir = ".\transferred-videos" # рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ
)
```

## ­ЪћД рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃГрЃарЃљ

### рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ: "рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљ рЃцрЃљрЃўрЃџрЃў"

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ:**
1. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ, рЃарЃЮрЃЏ `public/videos/` рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА
2. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ, рЃарЃЮрЃЏ рЃЋрЃўрЃЊрЃћрЃЮ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА

### рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ: "рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃўрЃАрЃљрЃА"

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ:**
1. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃЦрЃЋрЃЌ рЃАрЃљрЃЎрЃЏрЃљрЃарЃўрЃАрЃў рЃЌрЃљрЃЋрЃўрЃАрЃБрЃцрЃљрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃў
2. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃЦрЃЋрЃЌ рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃгрЃљрЃАрЃљрЃЎрЃўрЃЌрЃ«рЃљрЃЊ
3. рЃЊрЃљрЃ«рЃБрЃарЃћрЃЌ рЃАрЃ«рЃЋрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃЊрЃюрЃћрЃю рЃцрЃљрЃўрЃџрЃћрЃЉрЃА

### рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ: "PowerShell execution policy"

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ:**
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

## ­ЪЊі рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў

рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЊрЃљрЃўрЃюрЃљрЃ«рЃљрЃЋрЃЌ:

- РюЁ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃўрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ
- РЮї рЃЋрЃћрЃа рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃўрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ (рЃЌрЃБ рЃљрЃарЃўрЃА)
- ­ЪЊЂ рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў

## ­Ъј» рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў

### PowerShell рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў

```powershell

# рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ
```

### Node.js рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃЏрЃЮрЃЊрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ
